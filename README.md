# Task Master
**Co-author:**<br/>
[Justyna Gapys](https://github.com/justynagapys)<br/>

## Description
Application which lets add, edit and delete tasks. It includes also user authentication.<br/>
- Ionic, Angular.js,<br/>
- Firebase backend.<br/>
